# amap-react

**Using AMap with React in an easy way.**

![](https://jimnox.gitee.io/amap-react/img/logo-mini.png)

## 使用

**推荐使用 yarn。**

```bash
yarn add @amap/amap-react # 或 npm install --save @amap/amap-react
```

**@amap/amap-react 目前支持 AMap JSAPI 2.0。**
